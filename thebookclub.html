THE BOOK CLUB
March 2021 / Designed as part of an assignment for my User Interface class, The Book Club is a meeting space for book lovers of all genres, allowing users to find clubs related to their interests, participate in discussion forums, and join live events all within the app.

ABOUT
The assignment was to create a user interface for a mobile app and the options were to create one or a combination of a social network, matching service, or online live event platform. Color contrast ratio had to be at least 4.5:1. Created in Figma, click here to view the full project.
PREVIEW
Medium-Fidelity
I began by sketching ideas and then creating medium-fidelity prototypes to be critiqued by peers. A preview can be seen below. Originally, the plan was to make add a video-conferencing feature, but decided in the end to keep it voice-based after critiques found the flow to be too confusing. The homepage was also cleaned up in high-fidelity and a top menu was chosen over a consistent fixed bottom bar.

High-Fidelity
Below is a quick preview of the final interfaces. Given more feedback, the plan is to continue iterating and perfecting.

Splash Screen
Screen that shows when the app is loading.

Home
What the user sees when the app has opened.

Meeting Details
User views the meeting name, scheduled starting time/date, number of people who have joined, and options to remove from their meetings, view the hosting club page, and join live meeting.

Live Meeting
User views the current topic of discussion, who is speaking as shown in red, moderators, and other readers who have joined. Options to react with a thumbs up, to request to speak, or to leave are below.

Meetings Tab
User can view their meetings as well as recommended meetings based on their tags (set in their settings). User sees that they can search for meetings.

Meeting Search
User can search for meetings using a keyword.

Meeting Details
User views the meeting details and can add to their meetings or view the hosting club’s page.

Clubs Page
User views a club page that they’ve enrolled in and can see details about the club including: members, option to leave club, club description, relevant club tags, upcoming meetings, moderators, and option to view the discussion forum.

Discussion Forum
User can interact with other club members in the club's discussion forum by creating a new topic or replying to topics.
