VIDEOLAB
February 2021 / Designed as part of an assignment for my User Interface class, VideoLab is a streaming platform tailored to both creators and viewers, focused mainly on gaming but with categories for lifestyle and music topics as well. Creators can stream and/or upload edited videos, viewers can create playlists and watch multiple streams simultaneously using a dual viewer. View the full project on Figma.

Parameters included a size of 1920x1080, color contrast minimum 7:1, and general instructions were to create a media streaming platform with playlist options.

PREVIEW
Homepage

What users see when they log in, including creators they’re following and playlists on the side, plus ‘Your Feed’ with recent uploads and streams from creators they’re following.
Stream Page

Users can see other recent uploads from the creator or recent streams, plus options to open the stream in a dual player or add the stream to a playlist. Chat features general, subscriber-only, and private chats between users.
Dual Viewer

Allows the user to watch multiple live videos at one time.
